# Installation

## Prerequisites

- Git
- Python 3.11+
- Optional: LaTeX distribution for manuscript builds

## Clone

```bash
git clone git@github.com:alexanderholman/mp1-1-sigma3-112-repro.git
cd mp1-1-sigma3-112-repro
```

## Update

```bash
git pull --ff-only
```

## Remove

Delete the cloned directory when no longer needed.
